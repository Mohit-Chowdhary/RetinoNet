About Retina.AI

Advancing diabetic retinopathy screening through explainable artificial intelligence and automated quality assessment.

Our Mission

Diabetic retinopathy is a leading cause of vision loss worldwide, affecting millions of people with diabetes. Early detection is crucial, but manual screening is time-consuming and requires specialized expertise that isn't always available.

Retina.AI was developed to bridge this gap by providing healthcare providers with an AI-powered screening tool that combines clinical-grade accuracy with explainable decision-making and automated quality control.

Technology & Methodology

Deep Learning Architecture

Built on EfficientNet-B0, a state-of-the-art convolutional neural network optimized for medical imaging. The model was trained using transfer learning with ImageNet pre-trained weights, then fine-tuned on comprehensive diabetic retinopathy datasets.

Focal Loss Training

To handle class imbalance in DR datasets, we implemented Focal Loss instead of standard cross-entropy. This technique focuses learning on hard-to-classify samples and improves performance across all severity levels.

Hyperparameter Optimization

Used Optuna framework for automated hyperparameter tuning, optimizing learning rate, weight decay, focal loss gamma, and batch size. Final model selected based on validation set Quadratic Weighted Kappa score.

Explainable AI

Integrated Grad-CAM (Gradient-weighted Class Activation Mapping) to generate visual explanations for every diagnosis. This builds trust by showing clinicians exactly which retinal features influenced the AI's decision.

Quality Assessment Pipeline

Seven-metric quality evaluation system using OpenCV for blur detection (Laplacian variance), illumination assessment, contrast analysis, artifact detection, centering validation, resolution check, and color balance evaluation.

Clinical Validation

Model performance evaluated using industry-standard metrics: overall accuracy, per-class precision/recall/F1-score, confusion matrix analysis, and Quadratic Weighted Kappa for ordinal severity assessment.
