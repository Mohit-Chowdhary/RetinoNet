Explainability

Grad-CAM Visual Explanations

Our Explainable AI (XAI) system uses Gradient-weighted Class Activation Mapping (Grad-CAM) to generate intuitive heatmaps that highlight the specific regions of the retinal image that influenced the AI's diagnostic decision.

  • Visual heatmap overlays on original fundus images
  • Color-coded impact levels (Low, Medium, High, Critical)
  • Side-by-side comparison views
  • Identifies anatomical features: microaneurysms, hemorrhages, exudates
  • Builds trust and clinical confidence in AI decisions
Quality Control

Automated Image Quality Assessment

Before analysis, every fundus image undergoes a comprehensive 7-metric quality evaluation to ensure diagnostic reliability. Our system provides real-time feedback and actionable recommendations for image improvement.

  • Blur Detection: Laplacian variance method for sharpness analysis
  • Illumination Assessment: Automatic exposure evaluation
  • Contrast Evaluation: Image clarity measurement
  • Artifact Detection: Identifies eyelashes, dust, obstructions
  • Centering Validation: Optic disc positioning check
  • Resolution Check: Ensures adequate image size
  • Color Balance: Detects color cast issues
